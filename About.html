New User Guide: Getting Started

Hi! If you are a new first time Aurora User and this is your first time using Aurora you < b>must do the following immediately.

Step 1: Setting Up an account.

To make your device fully yours you should start by making an account. Just go to Settings > User > Set your name and pfp url > Click Save User Settings.

Step 2: Setting Up a password

This is the most important step in setting up your device. The first time you visit Aurora you are granted access without any password. From next on you will be asked for a password. If you don't setup a password now you will be locked up forever. So make your password by Settings > User > Change Device Password > Save User Settings.

New App Installation

Installing Applications in Aurora is very simple. Just-

  1. Open Shop: Locate the 'Shop' icon on your desktop or in the main menu.
  2. Search: Search your designated application.
  3. S1: Click the 'Download' button.
  4. S2: Copy the link to your clipboard.
  5. S3: Go to the 'Settings' app. Go to the 'Programs' tab. Scroll down to 'Install New Application'. Enter the app name, App source URL (The one you copied), and enter the font awesome icon according to your liking.
  6. S4: Finally click the 'Install App' button and done!

Downloaded apps can also be deleted from the 'Installed Applications' section from the 'Programs' tab in the 'Settings' app.
